ובנוסף הקישור בין צד הלקוח לשרת ושליחת בקשות בין שרת ללקוח אתגרו אותי מאוד.
Pip install virtualenv עבור התקנה לסביבה ווירטואלית:
Virtualenv env
Env\ scripts\activate
Pip install Django – התקנת   django
Django-admin startproject <name of project> - יצירת פרוייקט
Python manage.py startapp <name_of_app > - בניית אפליקצייה 
Python manage.py runserver – הרצת השרת
כדי לבנות טבלאות ב-DB תצטרך לכתוב את המודלים שלך בפרויקט Django ולהקליד:
python manage.py makemigrations
python manage.py migrate -> זה יצור טבלאות ב-DB.
הסרטונים מסבירים את העבודה עם Django 
ומכל סרטון לקחתי חלק ושילבתי ביניהם.
